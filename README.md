# NeuroGolf 2026

Kaggle notebooks for the NeuroGolf 2026 competition.

## 1. Notebook Index

- `notebooks/01_eda.ipynb`: Kaggle-ready EDA for ARC-style task JSON files, using `viridis` for charts and prefixed section headings.

## 2. Kaggle Usage

Upload or copy the notebook into Kaggle, attach the NeuroGolf 2026 competition/public data, and run top-to-bottom. The notebook discovers JSON files under `/kaggle/input` and writes summary CSVs to `/kaggle/working`.
